# ol3-ext
==============

Cool extensions for OpenLayers 3 (ol3).  
[View live examples online](http://viglino.github.io/ol3-ext/).

CSS popup, 
Font Awesome symbol renderer, 
charts for statistical map (pie/bar), 
layer switcher,
wikipedia layer, 
animations
